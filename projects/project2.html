Project 2: Enhancing E-commerce Product Search

In this project, we aimed to enhance the product search functionality for an e-commerce website to help users find products more efficiently.

Research Methods

  • Contextual Inquiry
  • Card Sorting
  • A/B Testing

Findings

The research revealed that users were struggling with the current search filters and found the search results to be irrelevant to their queries.

Solutions

We redesigned the search filters to be more intuitive and implemented a machine learning algorithm to improve search result relevance. This resulted in a 30% increase in user satisfaction and a 25% boost in sales.
